# PRODIGY_WD_05

## Weather Web Application

A responsive web app that fetches current weather information from OpenWeatherMap API based on user input or location.

---

### Features

- Search weather by city
- Displays temperature, humidity, condition, and wind speed
- Fully responsive and modern design
- Smooth and aesthetic UI

---

### Technologies Used

- HTML5
- CSS3
- JavaScript (Fetch API)
- OpenWeatherMap API
